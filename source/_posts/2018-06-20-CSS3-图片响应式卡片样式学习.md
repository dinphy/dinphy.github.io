网络上很流行的卡片式网页，尝试了一下，其实不难。

##### **案例分析**

- 容器 a 固定高和宽，设定背景颜色，加上阴影效果，卡片向左浮动，卡片之间设置间隔距离，设置图片内边距。
- 容器 b 文本居中，设置文本内边距。
- 图片于卡片中自动居中。

##### **展示效果**

![](https://raw.githubusercontent.com/dinphy/dinphy.github.io/master/images/555.PNG)

##### **案例展示**

```js
<!DOCTYPE html>
<html>
<head>
<title>测试卡片式图片</title>
<style>
	div.a {
	  width: 120px;
	  height: 90px;
	  background-color: white;
	  box-shadow: 0 4px 8px 0 rgba(0, 0, 0, 0.2), 0 6px 20px 0 rgba(0, 0, 0, 0.19);
	  float: left;
	  margin: 5px 5px;
	  padding: 10px 0 10px 0;
	}

	div.a > a img{
		margin: 0 auto;
		display: block;
	  }

	div.b {
	  text-align: center;
	  padding: 2px 10px;
	}
</style>
</head>
<body>

	<h2 style="text-align: center">响应式卡片</h2>

	<div class="a">
	  <a href="#"><img src="111.PNG"></a>
	  <div class="b">
		<p>卡片一</p>
	  </div>
	</div>

	<div class="a">
	  <a href="#"><img src="222.PNG"></a>
	  <div class="b">
		<p>卡片二</p>
	  </div>
	</div>
	<div class="a">
	  <a href="#"><img src="333.PNG"></a>
	  <div class="b">
		<p>卡片三</p>
	  </div>
	</div>
	<div class="a">
	  <a href="#"><img src="444.PNG"></a>
	  <div class="b">
		<p>卡片四</p>
	  </div>
	</div>
</body>
</html>
```
